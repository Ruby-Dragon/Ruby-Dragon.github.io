Rust To Do List

GNU Public Licence v3, 2022, Ruby-Dragon

A simple CLI to-do list written in Rust!

Useful for keeping track of tasks in a stylish and fast way. Best when used with Linux and a nice terminal theme.

This is how I learned Rust, which is pretty nice. I like lists apparently.

Use with a terminal that supports ANSI escape codes, CMD and Powershell do not, which makes the strikethrough not appear in completed tasks! Windows Terminal and most Linux terminals work correctly.

Program Information:

What is it?

This is a command line to-do list written in Rust that allows for easy tracking of tasks. It allows for deadlines and completed tasks will disappear after their deadlines. Works on Windows and Linux, should work on macOS.

It uses command line arguments to add tasks, remove tasks, and complete tasks. Usage is explained in the README.md file!

Github Link

Installation:

Install cargo and Rust if it is not already installed.

Clone repo to pc.

cd into folder.

use "cargo build --release"

Then the executable should be: ./target/release/rust_to_do_list

Add a bash alias for the executable, if you use bash, to allow for easy use!
