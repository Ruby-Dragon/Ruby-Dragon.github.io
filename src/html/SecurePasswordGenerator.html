Secure Password Generator

GNU Public Licence v3, 2022, Ruby-Dragon

Generate secure-ish passwords easily!

Not cryptographically secure, but better than password1!

This project was made to practice CPP and keep my skills up - and to work on when im bored.

Written in C++ because I decided learning Rust just for this was going to kill me, and I was done suffering with Java.

Program Information:

What is it?

This program is a command line app that allows for randomized password to be generated with one command. It is written in CPP and works on Windows, Linux, and should run fine on Mac.

It uses command line arguments to change options to allow for different sizes of password, and an option to use words instead of symbols. Usage is explained in its README.md file.

Github Link

Installation:

Clone repo to pc.

cd into folder.

use the command: "cmake" to compile the code.

Then the executable should be: ./SecurePasswordGenerator
