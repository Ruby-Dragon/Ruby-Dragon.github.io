PCPartList

GNU Public Licence v3, 2022, Ruby-Dragon

Keep track of your PC parts!

This CLI program tracks the cost and power consumption of your PC parts! Keep organised and stay within your budget, while also keeping track of your PC's power consumption! It saves logs of old builds, so you know what that one part that you can't remember where you got it came from!

I wrote this program in Java for practice, and also to try and make something genuinely useful.

Program Information:

What is it?

This is a command line app to keep track of your parts for a PC build. It allows for multiple saved lists, and calculates total cost and power draw. It can also save links, so you know where you got everything!

It uses an interactive console to add, remove, and edit parts, and also to create and delete lists. Usage is shown in the README.md file.

Github Link

Installation:

Install Java (latest version recommended)

Clone repo to pc.

cd into folder.

Run ./compile.sh script to compile source code.

Then the executable should be: ./builds/PCPartList.jar

Always keep the lists/ directory in the same directory as PCPartList.jar!

Run with java -jar ./builds/PCPartList.jar
