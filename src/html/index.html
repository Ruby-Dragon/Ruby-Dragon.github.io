This stylesheet set has multiple "Themes", which each look different. Currently, only the "Forest" theme is officially supported, and may be worked on more later, but custom themes can be made by editing the colors in the stylesheets.

Use this theme by linking the stylesheets into your own HTML and using the CSS classes defined there. Use the HTML files here as examples to guide your own site! You can make your own HTML or modify this code. I am not interested in making pages for very specific features or for signup/login, so you will have to make those yourself.

An important thing to keep in mind, is that the blur effect is not currently supported on Firefox (97, at time of writing). This may change in the future, but keep in mind when creating your own themes that the site needs to look good even without the blur effect.
