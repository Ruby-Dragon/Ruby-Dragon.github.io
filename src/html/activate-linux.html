Program Information:

What is it?

This the Activate Windows watermark ported Linux and macOS. It will not run on Windows.

This program is obviously a joke, but can be actually useful if you want a watermark on your screen on Linux for whatever reason. The watermark can be customised using command line arguments, which you can learn more about in ARGS.md. Add to your WM's or DE's startup script to have it display on login.

Github Link

Installation:

Clone repo to pc.

cd into folder.

use the command: "make" to compile the code.

Then the executable should be: ./bin/activate_linux or ./bin/activate_macos

If you use Arch Linux, then activate-linux is available from the AUR under activate-linux-git.

If you use OpenSuse, then activate-linux is available from the OBS under activate-linux.
