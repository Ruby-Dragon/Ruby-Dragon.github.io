Frosted Glass

GNU Public Licence v3, 2022, Ruby-Dragon

Modern website styling and layout.

Styles for making beautiful looking websites, without the hassle of writing your own styles!

All colors are declared as CSS variables, so editing colors is as easy as ctrl-c ctrl-v!

Very little JS use, almost everything is done in HTML and CSS!

Program Information:

What is it?

Frosted Glass is a bundle of stylesheets and JS scripts needed to make a nice modern website! This site uses the Frosted Glass styles and scripts.

It includes example HTML files that can be modified to suit your website. Colors can be changed by changing the colors defined in main.css to your own colors.

Custom CSS can be added, but is not necessary in most cases.
